# Task 04 — Portfolio Stress Test Simulator

## The Idea

A CLI tool that stress-tests any portfolio against **real historical market crashes** — answering the question every wealth manager should ask: *"Would this portfolio have survived 2008?"*

## Why This Matters for Timecell

Timecell's core promise is **"crash survival, runway, portfolio math — with the reasoning, not just the answer."**

This tool brings that promise to life by:
1. Replaying actual crash scenarios (2008 GFC, COVID-19, Crypto Winter 2022) against a user's portfolio
2. Computing post-crash value and remaining runway for each event
3. Using an LLM to generate actionable "lessons learned" from each scenario
